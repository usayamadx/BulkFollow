## Steps

### install
```
pip install requests requests_oauthlib

```

### create config.ini

```
touch config.ini
vi config.ini
```

set
```
[settings]
API_KEY = "Your API key"
API_SECRET_KEY = "Your API secret key"
ACCESS_TOKEN = "Your Access token"
ACCESS_TOKEN_SECRET = "Your Access token secret"
```
